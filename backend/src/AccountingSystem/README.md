# AccountingSystem — Backend

Questo progetto contiene il backend ASP.NET Core per il sistema di contabilità.


Per eseguire:
1. Aprire `src/AccountingSystem`.
2. Configurare connection string in `appsettings.json` o usare la default (LocalDB).
3. Eseguire `dotnet restore` e `dotnet ef database update`.
4. `dotnet run`.

API disponibili via /swagger.
